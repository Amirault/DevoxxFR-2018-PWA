Bienvenue !

Vous êtes sur une application développée par des étudiants de Polytech' Paris-Sud dans le cadre de la matière "Algorithmes pour le Web".

L'idée du projet est de créer une Progressive Web App. Nous avons donc choisi de créer une application autour de l'API de la Devoxx de Nicolas Martignole. Vous pouvez donc utiliser cette application pour :

  • Consulter le planning des 3 jours ainsi que le détail des événements
  • Bookmarker les conférences que vous ne voulez pas louper et les retrouver dans un planning dédié (en cache sur votre appareil)
  • Consulter la liste des intervenants afin d'en apprendre d'avantage sur les différents speaker

... Et le tout même sans internet (merci les services-workers !). N'hésitez pas installer l'application sur votre téléphone (si votre navigateur vous le propose) afin de faciliter vos 3 jours à la Devoxx ;)

Pour finir, n'hésitez pas à nous faire un retour sur cette application ou même contribuer via son dépôt Gitlab. Toutes les remarques sont bonnes à prendre :)
