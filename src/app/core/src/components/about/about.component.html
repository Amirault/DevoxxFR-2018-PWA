Bienvenue !

Vous êtes sur une application codée par des étudiants de Polytech' Paris-Sud dans le cadre de la matière "Algorithmes pour le Web".

L'idée du projet est de créer une Progressive Web App. Nous avons donc choisis de créer une application autour de l'API de la Devoxx de Nicolas Martignole. Vous pouvez donc utiliser cette application pour :

  • Consulter le planning des 3 jours ainsi que le détails des évenements
  • Bookmarker les conférences que vous ne voulez pas louper et les retrouver dans un planning dédié (en cache sur votre appareil)
  • Consulter la liste des intervenants

... Et le tout même sans internet (merci les services-workers). N'hésitez pas installer l'application sur votre téléphone (si votre navigateur vous le propose) afin de faciliter vos 3 jours à la Devoxx ;)

Pour finir, n'hésitez pas à nous faire un retour sur cette application ou même contribuer via son dépôt Gitlab. On est très intéréssés par vos remarques !
